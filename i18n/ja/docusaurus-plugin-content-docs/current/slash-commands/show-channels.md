# /show-channels

## 説明

すべての翻訳チャンネルのリストを表示します。

```command
/show-channels
```

:::info もう一度、翻訳チャンネルとは何ですか？
翻訳チャンネルは、言語が設定されているチャンネルのことです。
:::

:::note
Discord のメッセージ文字制限は 2,000 文字です。 リストが Discord の制限を超える大きさの場合、リストは埋め込みメッセージ (embed) として送信されます。
:::
