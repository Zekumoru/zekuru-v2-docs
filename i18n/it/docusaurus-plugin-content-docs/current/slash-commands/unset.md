# /unset

## Descrizione

Rimuovi la lingua impostata su un canale.

```command
/unset <channel: opzionale>
```

## Opzioni

### channel: opzionale

Il canale da cui rimuovere la lingua. Se non specificato, viene rimossa la lingua dal canale corrente in cui viene utilizzato il comando.
